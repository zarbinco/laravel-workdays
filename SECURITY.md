# Security Policy

If you discover a security issue in `zarbinco/laravel-workdays`, please do not open a public issue with exploit details.

Report the issue privately through the GitHub repository security advisory flow, or contact the maintainers through the repository owner profile.

Please include:

- A clear description of the issue.
- Steps to reproduce when possible.
- Affected versions or commit references.
- Any suggested mitigation.

We will review reports as quickly as practical and coordinate a fix when needed.
